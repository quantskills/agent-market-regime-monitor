# Agent Boundary / 使用边界

这个仓库是 QuantSkills 组织的研究与盯盘 Agent，不是自动交易系统。

## 它会做什么

- 使用 Pandadata 或用户提供的数据形成研究判断。
- 生成图文报告、观察清单、决策矩阵和复盘模板。
- 给出支持证据、反证条件和下一步观察动作。
- 帮助研究员、交易者或 AI Agent 继续分析。

## 它不会做什么

- 不连接券商、交易终端或下单接口。
- 不执行订单。
- 不输出无条件买卖指令。
- 不承诺收益或胜率。
- 不把单一指标解释为完整交易系统。

## 用户需要自己决定什么

- 是否把报告纳入自己的研究流程。
- 是否需要补充更长窗口、更多品种或人工事件解释。
- 是否把观察结果转化为交易计划。
- 仓位、止损、风控和执行规则。

## 给 AI Agent 的要求

调用本 Agent 时，应优先读取 `AGENT.md`、`README.md` 和 `references/`。如果数据不足，应输出“证据不足”或“需要复核”，不要补造结论。
