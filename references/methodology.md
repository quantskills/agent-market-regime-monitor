# Methodology / 方法说明

本 Agent 用 Pandadata 的融资余额、龙虎榜事件热度和历史波动率证据，观察市场环境是否处于热度扩张、风险扩张或混合观察状态。它适合用于晨会、复盘和研究样本筛选。

## 核心问题

当前市场环境是否支持继续扩大研究样本，还是应该先收紧风险观察？

## 使用的数据

- Pandadata methods: `get_margin`, `get_lhb_list`, `get_option_underlying_volatility`
- 主要产物：`outputs/live/regime_scorecard.json`
- 关键图表：
  - `outputs/live/charts/funding_balance.png`
  - `outputs/live/charts/event_reason_distribution.png`
  - `outputs/live/charts/scorecard_metrics.png`

## 判断逻辑

1. 观察融资余额是否扩张或回落。
2. 统计龙虎榜事件原因和样本密度。
3. 使用历史波动率辅助判断风险是否升温。
4. 当融资余额、事件热度和波动率共同上升时，市场处于热度或风险扩张观察区。
5. 当热度扩张但承接质量未确认时，结论应保持为观察状态，而不是交易指令。

## 输出解释

- `热度扩张观察`：市场样本和资金热度较高，可以扩大研究样本，但需要同步观察拥挤风险。
- `风险扩张观察`：热度和波动同时升高，应优先关注风险预算。
- `混合观察`：证据方向不一致，需要等待更多数据确认。

## 适合继续追问的问题

- 热度扩张是否伴随波动率失控？
- 龙虎榜事件是扩散到更多样本，还是集中在少数标的？
- 融资余额变化是否支持继续扩大研究范围？
- 哪些条件会推翻当前市场状态判断？

## 使用边界

市场状态不是交易信号。本 Agent 用于环境判断和研究优先级排序，不替代择时模型、仓位系统或执行规则。
