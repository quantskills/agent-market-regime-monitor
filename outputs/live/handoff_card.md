# 市场状态盯盘 Agent 交接卡

## 当前回答的问题

当前市场环境是否支持继续扩大研究样本，还是应该先收紧风险观察？

## 当前状态

- 状态标签：`heat-expansion-watch`
- 风险等级：`中高关注`
- 主要观察焦点：融资余额、龙虎榜事件原因分布、历史波动率。

## 现在应该怎么用

把市场状态作为研究环境标签，结合融资余额、龙虎榜热度和波动率决定研究优先级。

## 升级与降级

- 升级观察：融资余额、事件热度和承接质量继续同向改善。
- 降级观察：热度仍高但融资余额回落、波动率升高或承接转弱。

## 边界

本 Agent 只提供研究、盯盘和复盘输入，不负责交易执行。
